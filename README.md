# Golebiewski-et-al-AJHG-2026

This repository contains all necessary code to reproduce data in Golebiewski, Anna K.; Stolze, Lindsey K.; Vazquez, Valentina V.; Mehrabi-Yazdi, Alhan; Careaga, Cecilia M.; and Romanoski, Casey E. (2026). An atlas of endothelial RNA splicing catalogs effects of IL1B and identifies an alternative PROCR isoform with genetic links to pleiotropic vascular disease. The American Journal of Human Genetics. (accepted)_

With questions please contact the corresponding author: cromanoski@arizona.edu
